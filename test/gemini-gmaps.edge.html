Gemini Google Maps

All map types require an apiKey be passed in the initializer object.

Usage
G('.selector').gmaps({ apiKey: "some-key-from-google-cloud-console" });

Embedded → Default Type

Usage
G('.selector').gmaps({ 
  
  locations: [ { title?, address?, lat,lng } ... ] 
  
  // OR  -------------------------------------------

  embedQuery: '2520 Haines Road, Mississauga, ON'

});

Embedded map types are the default map type.

You need to pass an embedQuery, or a locations array with an address or title to the G('.selector').gmaps() initializer object.
The embedQuery, address, or title can be an address (2520 Haines Road) or business name (Carpages.ca).

Limits

Unlimited loads / month

Static

Usage
G('.selector').gmaps({ locations: [{ title, lat,lng }], type: 'static' });
Limits

100,000 loads / month

Caveats

A static map image cannot be larger than 640x640.

Dynamic

Usage
G('.selector').gmaps({ locations: [{ title, lat,lng }], type: 'dynamic' });
Limits

28,000 loads / month
